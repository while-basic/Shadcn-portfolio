# MIDI Saber

A custom MIDI drum pad interface designed to streamline the level mapping process for the popular virtual reality game Beat Saber. This tool enables quick and intuitive creation of beat maps using MIDI input.

## Features

- MIDI drum pad interface
- Quick level mapping capabilities
- Real-time beat mapping
- Custom map creation
- Integration with Beat Saber

## Technologies Used

- JavaScript
- Unity
- MIDI Integration
- Game Development Tools
- WebMIDI API
